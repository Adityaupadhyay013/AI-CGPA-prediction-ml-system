It is an ml system that predicts the current CGPA of student on the basis of previos sem CGPA , Social media usage , Gaming hours , Sleeping hours and some other inputs. 
This project also involves a frontend and a backend. 
It is a deployed project as hugging face space. 
This results of Student CGPA prediction ml model sent to ai agent ehich interprets these results and show detailed analysis of these results along with some tips to improve
CGPA , derives some useful insights for them. 
This ml model also provides features responsible for the prediction of a particular value of CGPA.
